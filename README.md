# Youtube Downloader CLI

Script to download youtube videos in linux using CLI.  
- Provide url of video
- Indicate temporal folder to download video or use DEFAULT
- Choose if want to convert to mp3 format
- Choose if it should be copied to cloud folder (current date for folder's name)
- At the end files are moved to local folder (current date for folder's name)
---

## Prerequisites

Install Deno - [Link](https://docs.deno.com/runtime/getting_started/installation/)

```bash
curl -fsSL https://deno.land/install.sh | sh
```

### Test installation

```bash
deno --version
```

### Create venv and source it
```bash
python3 -m venv venv
source venv/bin/activate
```

### Install requirements.txt
```bash
pip install -r requirements.txt
```
---


## Run
```python
python3 main.py
```
Follow instructions.
